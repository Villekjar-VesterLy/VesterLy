# VesterLy gæsteguide

Færdig Vite/React demo-app til VesterLy.

## Deploy på Vercel

1. Upload denne mappe til et nyt GitHub repository.
2. Gå til https://vercel.com/new
3. Vælg repository.
4. Framework preset: Vite
5. Build command: npm run build
6. Output directory: dist
7. Tryk Deploy.

## Lokal test

npm install
npm run dev
